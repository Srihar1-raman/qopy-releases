# qopy releases

## Downloads

- **macOS**: `qopy.dmg`
- **Windows**: `qopy-windows.exe`

## Windows requirement (OCR)

The Windows build uses **Tesseract OCR**, which must be installed separately.

- Download and install Tesseract from: https://github.com/UB-Mannheim/tesseract/wiki
- Then open qopy **Settings** and set the `tesseract.exe` path if it is not auto-detected.
